#  Gestion Cabinet Dentaire (Terminal)

##  Description
Application en langage C développée pour gérer un cabinet dentaire via le terminal.  
Permet la gestion des patients et des rendez-vous avec suivi simple.

##  Objectif
- Ajouter et afficher les patients  
- Supprimer un patient  
- Prendre et afficher les rendez-vous  

##  Fonctionnalités

###  Gestion des patients
- Ajouter un patient (Nom, Prénom, Téléphone)  
- Afficher la liste des patients actifs  
- Supprimer un patient (marqué comme inactif)  

###  Gestion des rendez-vous
- Ajouter un rendez-vous pour un patient existant  
- Afficher tous les rendez-vous avec ID patient, date et statut  

## Technologies utilisées
- Langage C  
- Tableaux de structures  
- Interface en mode terminal  

##  Structure du projet
- `main.c` : fichier principal contenant le programme
